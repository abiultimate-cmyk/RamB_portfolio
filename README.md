# Ramkumar Balakrishnan Portfolio

This is a static website portfolio for Ramkumar Balakrishnan, showcasing personal projects and social media links.

## Features

- Minimalist design with a black background and a blue/white color scheme.
- Animated browser tab title that cycles through the text "Ramkumar Balakrishnan" with color transitions.
- Links to social media profiles including Instagram, Facebook, Twitter, and IMDB.

## Setup Instructions

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd ramkumar-portfolio
   ```

3. Install the dependencies:
   ```
   npm install
   ```

4. Start the development server:
   ```
   npm start
   ```

5. Open your browser and go to `http://localhost:3000` to view the portfolio.

## Technologies Used

- React.js for building the user interface.
- CSS for styling the components.
- Custom hooks for managing animations.
